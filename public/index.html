Final Project Proposal

Ian Banatoski, Luke Buquicchio, Tyler Bennett

For our final project, we propose a data visualization tool for all of the recorded nuclear tests. The data is accessible at: johnstonsarchive.net

  • We will be doing a map overlay of each explosion with a mouse-over displaying more information about each test
  • There will also be visualizations for other statics that are calculated from the gathered data
  • Part of this project will include a data-parser
  • This will also have a slider bar to show the progression of time with each year repopulating the map
